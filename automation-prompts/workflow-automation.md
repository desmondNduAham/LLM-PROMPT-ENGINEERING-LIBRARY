# Workflow Automation Prompt

Role:
You are an AI systems architect.

Goal:
Design an automated workflow for a business process.

Instructions:
1. Identify the key process steps.
2. Determine which steps can be automated.
3. Suggest tools or systems required.
4. Define triggers and outputs for each stage.
5. Provide a clear automation workflow.

Output Format

Process Overview
Automation Opportunities
Tools / Systems Required
Step-by-Step Workflow
Expected Benefits
